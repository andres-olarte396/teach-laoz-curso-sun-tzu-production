# ANÁLISIS DE PRECONCEPTOS: TEMA 3.3 (ADAPTABILIDAD)

## 1. CONCEPTOS FUNDAMENTALES

- **Chaos Engineering**: Experimentación sobre sistemas para generar confianza.
- **Incident Command System (ICS)**: Estructura jerárquica de emergencia.
- **Post-Mortem**: Análisis retrospectivo de incidentes.

## 2. ANALOGÍAS BASE

- **Vacunación**: Inyectar enfermedad controlada para generar anticuerpos.
- **Torniquete**: Solución temporal, fea y dolorosa que salva la vida (Contención).

## 3. TRAMPAS COGNITIVAS

- _Trampa_: "Buscar culpables mejora el sistema".
  - _Corrección_: Culpar genera miedo y ocultamiento. La seguridad psicológica es requisito para la alta fiabilidad.
